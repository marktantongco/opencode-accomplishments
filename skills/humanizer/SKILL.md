# Humanizer

## context

Strip AI writing patterns for human-like copy — anti-AI detection, natural voice restoration, and style matching. Makes AI-generated content indistinguishable from human-written text. This skill provides content creation frameworks that balance quality, consistency, and performance across channels and formats.

## instructions

### Step 1: Understand Content Requirements

1. Identify content type — blog post, social media, documentation, or marketing copy
2. Define target audience — technical level, context, and motivation
3. Establish tone and voice — brand guidelines, formality level, personality
4. Set content goals — awareness, engagement, conversion, or education

### Step 2: Apply Content Framework

1. **Research** — Gather source material, keywords, and competitive context
2. **Structure** — Create outline with clear hierarchy and flow
3. **Draft** — Write content following brand voice and format requirements
4. **Optimize** — Refine for readability, SEO, and platform constraints

### Step 3: Generate Content

```markdown
## Content Brief

### Target Audience
Audience description with demographics and psychographics

### Key Messages
1. Primary message
2. Supporting message
3. Call to action

### Content Structure
- Hook: attention-grabbing opening
- Body: main content sections
- Close: summary and CTA

### SEO Elements (if applicable)
- Primary keyword: target keyword
- Secondary keywords: related terms
- Meta description: 155 characters
- H1: optimized heading

### Platform Adaptations
- Platform 1: adaptation notes
- Platform 2: adaptation notes
```

### Step 4: Quality Review

- [ ] Content matches brand voice and tone
- [ ] All claims are supported and accurate
- [ ] Formatting is clean and scannable
- [ ] Call-to-action is clear and compelling
- [ ] Platform-specific requirements are met

## constraints

- NEVER publish unverified claims or statistics without sources.
- NEVER sacrifice readability for keyword density.
- NEVER use the same content across platforms without adaptation.
- ALWAYS match tone to audience and platform.
- ALWAYS include a clear call-to-action.
