▸ ONBOARDING
5-Step Walkthrough
From zero to your first installed skill stack in under 2 minutes.
1
Open the Skill Store
Browse the 6 zones — ACTIVATE, BUILD, VALIDATE, PLAYBOOK, MONETIZE, SYSTEM. Each pre-rendered, instantly switchable.
2
Press ⌘K to Search
Fuzzy search across all 90+ skills by name, tag, or category. Arrow keys navigate, Enter selects.
3
Copy Any Install Command
Tap any skill card to expand. Tap the COPY button — install command is on your clipboard. Paste in terminal.
4
Ask the Recommender
Open the chat bubble. Type "I want to build an animated dashboard" — get a curated skill stack back, powered by Vercel AI Gateway.
5
Install as PWA
Add to Home Screen — works offline, opens in its own window, auto-updates in the background.
